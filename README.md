 # 📚 C# Fundamentals - Öğrenme Serüveni

Bu repository, C# dilini en temel konulardan başlayıp ileri seviye kurumsal mimarilere (Enterprise Architectures) kadar öğrenme sürecimi adım adım belgelediğim bir çalışma alanıdır. 

Sıfırdan başlayıp nesne yönelimli programlama, hata yönetimi ve dosya işlemlerini kavrayarak ilerliyor; bir yandan da kendi ERP sistemimi yazacak altyapıyı kuruyorum.

## 📂 Klasör Yapısı ve İlerleme

### ✅ Week 01 - Basics (Tamamlandı)
C# dilinin temel yapı taşları, karar yapıları ve döngülerle algoritmik düşünme becerisinin kazanılması.
- [x] Day 01: Variables, Data Types, Type Conversions
- [x] Day 02: If-Else, Switch-Case, Ternary Operator
- [x] Day 03: For, While, Do-While, Foreach
- [x] Day 04: Arrays (Diziler)
- [x] Day 05: Methods (Metodlar, Overloading, Ref) & "Matematik Kütüphanesi Mini Projesi"
- [x] Day 06: Weekly Project - "Sayı Tahmin Oyunu"

### ✅ Week 02 - Collections & Core Concepts (Tamamlandı)
Uygulamalarda veriyi geçici (RAM/Koleksiyonlar) ve kalıcı (Dosya/FileIO) olarak yönetme, veri tipini güçlendirme.
- [x] Day 01: String Manipulations (Split, Substring, Replace vs.)
- [x] Day 02: Collections (List, Dictionary, LINQ Temelleri ve İç içe koleksiyonlar)
- [x] Day 03: Exception Handling (Try-Catch, Throw, ArgumentOutOfRange)
- [x] Day 04: File IO (StreamWriter, ReadAllLines, AppendAllText ile Kalıcı Kayıt / Loglama)
- [x] Day 05: Enum, Tuple, Nullable (Tip güvenliği ve çoklu veri dönüşü)
- [x] Day 06: Weekly Project - "Kişi Rehberi Uygulaması" (CRUD, Filtreleme, Dosya Entegrasyonu)
- [ ] Day 06.1: Challenge Project - "Mini Depo Yönetim Sistemi"
