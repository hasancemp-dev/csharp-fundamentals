# 📚 C# Fundamentals

C# dilini temelden ileri seviyeye kadar öğrenme sürecimi belgeleyen bir repo.

## 📂 Yapı

```
Week01-Basics/       → Değişkenler, kontrol yapıları, döngüler, diziler, metodlar
Week02-Collections/  → String işlemleri, koleksiyonlar, hata yönetimi
Week03-OOP/          → Sınıflar, kalıtım, polymorphism, abstraction
Week04-Advanced/     → Generics, LINQ, async/await, dosya işlemleri
```

## 🛠️ Teknolojiler

- **Dil:** C# (.NET 9)
- **IDE:** Visual Studio 2022

## 🚀 Nasıl Çalıştırılır

```bash
cd Week01-Basics/Day01-Variables
dotnet run
```
